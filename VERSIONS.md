### Verify checksums

Verify checksums (SHA-1):

```sh
sha1sum -c NGLEmp.sha1
```

Correct answer:

```
NGLEmp.8bx: OK
painter.cfg: OK
```

**Don't use NGLEmp release if checksums mismatch!**
