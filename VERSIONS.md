# Verify checksums

## Verify BLAKE3 checksums

```sh
b3sum -c NGLEmp.b3
```

Correct answer:

```
NGLEmp.8bx: OK
painter.cfg: OK
```

## Verify SHA-1 checksums

```sh
sha1sum -c NGLEmp.sha1
```

Correct answer:

```
NGLEmp.8bx: OK
painter.cfg: OK
```

**Never use NGLEmp release if checksums mismatch!**
